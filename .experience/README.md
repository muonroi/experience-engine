# Experience Engine v3.0

AI agents that learn from mistakes, not just store facts.
Works with Claude Code, Gemini CLI, Codex CLI, OpenCode.

## Quick Start (5 minutes)

```bash
# Clone any repo that includes .experience/
git clone <repo>
cd <repo>

# Run setup — interactive menu guides you through
bash .experience/setup.sh
```

Setup will ask you to choose:
- **Vector store**: Qdrant Cloud (free) / Local Docker / VPS tunnel
- **AI provider**: OpenAI / Gemini / Claude / DeepSeek / SiliconFlow / Ollama (free)

That's it. Brain installs to `~/.experience/`, hooks wire to all agents automatically.

## How It Works

```
YOU write code with any AI agent (Claude, Gemini, Codex, OpenCode)
  │
  ├─ BEFORE every Edit/Write/Bash → hook queries brain
  │   └─ "Have I seen this mistake before?"
  │   └─ If yes → injects warning into agent context
  │
  └─ AFTER every response → hook extracts lessons
      └─ Detects retry loops, error-fix patterns
      └─ Stores structured Q&A in vector DB
      └─ Next session: agent is slightly smarter
```

## Architecture

```
~/.experience/
  experience-core.js   ← brain (embed + search + extract + store)
  interceptor.js       ← hook wrapper (fires before tool calls)
  stop-extractor.js    ← hook wrapper (fires after each response)
  config.json          ← your settings (generated by setup.sh)
```

## Bootstrap Brain Instantly

Instead of waiting for organic learning, seed from existing rules:

```bash
# Seed from Claude memory files
node .claude/hooks/experience-bulk-seed.js --memory-dir ~/.claude/projects/<project>/memory

# Seed from any directory of markdown rule files
node .claude/hooks/experience-bulk-seed.js --memory-dir /path/to/rules
```

## Supported Providers

| Embedding | Brain (extraction) |
|-----------|-------------------|
| Ollama (nomic-embed-text) | Ollama (qwen2.5:3b) |
| OpenAI (text-embedding-3-small) | OpenAI (gpt-4o-mini) |
| Gemini (text-embedding-004) | Gemini (gemini-2.0-flash) |
| VoyageAI (voyage-code-3) | Claude (haiku) |
| | DeepSeek (deepseek-chat) |
| | SiliconFlow (any OpenAI-compatible) |

## Reconfigure

```bash
EXP_RESET_CONFIG=1 bash .experience/setup.sh
```

## Non-Interactive Setup (CI/scripts)

```bash
EXP_QDRANT_URL=http://localhost:6333 \
EXP_QDRANT_KEY=your-key \
OPENAI_API_KEY=sk-... \
bash .experience/setup.sh
```

## Requirements

- Node.js 20+
- One of: Docker, Qdrant Cloud account, or VPS with Qdrant
- One of: Ollama (free), or API key for any supported provider
